# Typography

<Unstyled>
  Typography is defined using classes and CSS variables, that could be find in [text.css](/src/styles/text.css) stylesheet and listed in the [reference table](#sizes-and-weights---reference-table) below.

By default, `.body-3` styles are applied to the `<body />` element. Using CSS universal selector
`font-size: inherit;` and `font-weight: inherit;` rules are applied to all elements. This allow to
change styles of all children just by adding necessary typography class to the parent element.

</Unstyled>

## Sizes and weights

There are 3 available text weights: `400`, `500` and `600`. In case, when the `font` rule is being
modified within `<style />` tag, shorthand syntax might be used to assign font weight as well:
`font: var(--body-3) / 500;`. NOTE: `font` rule modification should be done only when implementing
adaptive design by changing default-case styles, e.g. inside `:global(.tablet)` selector.

### Examples

<Unstyled>

```html
<div class="body-1 txt-b">
  Bitcoin

  <!-- font (body-1) inherited, but weight is changed -->
  <span class="txt-r">[bitcoin]</span>

  <!-- font is change, but weight is inherited -->
  <span class="caption">(BTC)</span>
</div>
```

```html
<div class="body-1 txt-b">
  Bitcoin

  <!-- font and weight are changed using font style rule -->
  <span id="ticker">(BTC)</span>
</div>

<style>
  :global(.desktop) #ticker {
    font: var(--caption) / 400;
    /* font =  size variable / weight */
  }
</style>
```

</Unstyled>
